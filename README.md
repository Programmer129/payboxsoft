## PayBox Soft

### Requirements

- JDK 1.8
- Node
- npm

### Usage

#### Back End

1) Go to `payboxsoft` directory

2) Run `gradle build`

3) Run `gradle bootRun`

##### Note: no need any DB configuration it uses in-memory H2
##### Note: Run tests with `gradle test`

#### Front End

1) Go to `payboxsoft-ui` directory

2) Run `npm install`

3) Run `ng serve`
